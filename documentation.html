Ideaz.cloud

Documentation

Welcome to ideaz.cloud! Below is a walkthrough for setting up and using the interactive discussion web application.

About this project

Ideaz.cloud is a discussion web application of sorts, which uses PHP, SQL, HTML, CSS, AJAX, Javascript, jQuery, and Twitter Bootstrap. It is much like an online forum in that it allows one to register an account and create threads (also known as "ideaz"), which are sorted in categories that users can freely create. Additionally, users are able to upvote threads; customize their "About Me" page; upload an avatar; and share videos, images, and files.

Let's Get Started!

1

Download and extract the .zip file.

2

Upload the contents of the /ideaz directory into your server (such as CS50's IDE).

Note: If using CS50's IDE, be sure to use apache50 start on the innermost /ideaz folder.

3

Create a new database (preferably titled "ideazcloud"). Import the contents from the ideazcloud.sql file into your newly-created database.

4

Navigate to the uploaded /includes/config.php file and enter your database information (host, username, password, and database name). The database name will likely be "ideazcloud".

5

Important: Be sure you have allowed file uploads on your server. You can do so by adding the following to your php.ini file:

file_uploads = On

(Optional) To ensure proper redirect to the 404 error page, add the following to your .htaccess file:

# 404 Not Found
ErrorDocument 404 /not_found.php

You're all set!

Usage

Administrator Account

The administrator username is admin, whose password is also admin. Using this account, you can edit and delete any thread or post. Want to create your own admin account? Log out and click the "Register" button in the navigation bar. A modal should pop up and give you basic user information to fill out. To give a user administrator privileges, you must go to the "users" table in your database and set the user's "is_admin" column to 1 (default is 0). It should be noted that all users are able to edit and remove their own posts and threads.

Categories

In order to create a thread, you must first be logged in. Then, navigate to the "Categories" page by clicking on its corresponding tab. Here, you will find the letters of the alphabet and various existing categories. If you cannot find a category pertinent to your idea, then simply create a new one! Once you have done so, you can find your category by clicking on the pill/tab that corresponds to the first letter of your newly-created category. Click on it and head to its category page.

Ideaz!

Here is where you will be able to create a new thread (idea). If logged in, simply click the "New Idea" button, enter a title for your thread, and some content. Notice how the body is much like a rich text editor. It turns out that you can upload files and images directly from this textarea. Additionally, you can simply copy and paste image or Youtube video links to the textbox, and they will automatically be converted to a visual representation of the image or video.

Once you are done, click "Post". You should now see your thread in the category's page.

Posts

Now that you have created your thread, click on it to arrive at its idea page. This is where the magic happens. You can upvote an idea by clicking on the lightbulb icon to the left of the thread title in the main thought bubble. This gives the thread an "Ideaz" point. Threads with the most ideaz will show up on the front page, which happens to be the "Big Ideaz" page.

Of course, the thread page is also where you can have a live discussion with others. If logged in, you should be able to see the text editor below your thought bubble. Add some content--text, images, files, videos--and click "Post". Without refreshing the page, your message should be displayed in the thread. Recent posts will show up closer to the top of the page, along with the user's avatar and username. To prevent clogging the page with a bunch of posts, infinite scroll has been implemented, which loads more posts when you scroll down. Of course, this only works when there are more than 5 posts in the thread.

User CP

Speaking of avatars, you may have noticed the default anonymous avatar next to your username. You can change this by clicking on the "User CP" link on the navigation bar beneath the website logo (next to "Logout"). You should now be at your user control panel. First thing's first: the "About Me" section. This is what others will see when they click on your username to view your profile. Notice how it uses the same rich text editor as on other pages. This means you can use text, videos, files, and images to express your interests. In the future, I plan on allowing one to customize their profile pages by editing the CSS and HTML.

Below the "About Me" section are the sections for allowing password and email changes. All of these sections are optional and can be left blank if one wishes to update only a particular aspect of their account/profile.

Finally, if you scroll all the way down, you will see the "Change Avatar" section. Here, you can upload an image that is less than or equal to 250 x 250 pixels. There is a dropzone, which gives users the option to drag and drop an image from their computer. Be sure your avatar is in .jpg, .jpeg, .gif, .png, or .pjpeg format!

New Ideaz

Now that we've covered the nitty-gritty of the website, we should take a look at the rest of Ideaz.cloud's features. The "New Ideaz" page is where new threads will be shown. The most recent ones will be displayed first.

Users

Here is where one can find other users. Clicking on a user's avatar or username will direct you to their profile, where you can see their username, post count, avatar, and information entered in the "About Me" section.

With that, we are finished!
